# DADK

DragonOS 应用开发工具包

DragonOS Application Development Kits

## 简介

DADK是一个用于开发DragonOS应用的工具包，设计目的是为了让开发者能够更加方便的开发DragonOS应用。

### DADK做什么？

- 自动配置libc等编译用户程序所需的环境
- 自动处理软件库的依赖关系
- 自动处理软件库的编译
- 一键将软件库安装到DragonOS系统中

### DADK不做什么？

- DADK不会帮助开发者编写代码
- DADK不提供任何开发DragonOS应用所需的API。这部分工作由libc等库来完成
